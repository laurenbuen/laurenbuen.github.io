# Taxi Rank Finder

This app provides an efficient solution for people to easily locate nearby taxi ranks. The  <a href="https://open-data-bristol-bcc.hub.arcgis.com/datasets/taxi-ranks/explore?location=51.457499%2C-2.600250%2C14.14&showTable=true">dataset<a> used consists of the taxi ranks' locations, the number of spaces available for each taxi rank and each taxi rank's operating time.
